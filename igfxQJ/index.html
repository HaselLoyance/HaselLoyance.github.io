technical name: igfxQJ Module
real name: YOLosu!bot
version: v1.5-alpha


YOLosu!bot is a bot for a rhythm game osu! released in 2007 by peppy. Its gameplay is based on Osu! Tatakae! Ouendan. Although there are four different game modes (osu!standard, Taiko, Catch the Beat, osu!mania), this bot is only for osu!standard.

Setup:

  1. Download a bot.
  2. Extract the archive to an empty folder.
    3. OPTIONAL: You may run any md5 checker to validate files.
  3. Rename "osu!bot.exe" to something not suspicious.
  4. Run it.

Usage:

  1. Press any key to bring up a map selection window.
  2. Pick any map file with the extension ".osu"
  3. Set up your settings by using commands and hitting enter after each one. Full list of commands:

    4. Commands that specify mods (for more human-like movement):
    1. >: HR - enables/disables Hard Rock
    2. >: DT - enables/disables Double Time
    3. >: FL - enables/disables Flashlight
    4. >: HD - enables/disables Hidden

    Commands that specify accuracy:
    1. >: 100 <amount> - specifies amount of 100's
    2. >: 50 <amount> - specifies amount of 50's
    3. >: 0 <amount> - specifies amount of misses

    Command for the style of tapping:
    1. >: tap - cycles through style of tapping

    Styles of tapping:
    1. Singletap - always uses only one button while botting
    2. Smart alternate - uses one button on slow parts and two on fast parts
    3. Alternate - uses both buttons one after another

    Control commands:
    1. >: build - generates the path of the cursor and timing of tapping before starting the map
    2. >: start - after this command bot will await for a one "z" input on the first note. Then your cursor will move automatically
    3. >: change - resets all builded parameters and brings up a map selection window to select a new map
    4. >: exit - safely exits the bot. PLEASE USE THIS INSTEAD OF JUST CLOSING THE PROGRAM
    5. >: HWID - shows your HWID

  4. Type "build" command after setting up all parameters
  5. After building process is done type "start" command
  6. Switch to osu! window
  7. Start the map
  8. YOU MUST CORRECTLY HIT THE FIRST NOTE USING "z" INPUT
  9. If you want to stop the bot press "y" or else just play the map to the end
  10. If you want to move the cursor during beak times - you are welcome to do so as long as you will stop moving in the same position where you started

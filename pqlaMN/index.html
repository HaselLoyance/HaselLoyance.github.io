Features

Image encryption

This library has our own image encryption/decryption algorithms. The greatest thing about it is that you can use your own key phrase for this to work. Works with any common image format. Does not correctly support alpha layer in *.png files. You'll have to install GraphicsMagick for this to work

Text encryption

This library has our own text encryption/decryption algorithms. The greatest thing about it is that you can use your own key phrase for this to work. Works with any files that have raw text (*.txt, *.html, *.css, *.cpp etc.)

Console costumization

Now you can set color or size of the console only with one line.

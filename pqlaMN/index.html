real name: YOLib
version: alpha v1

YOLF is a Windows library with the common functions, that we use in our projects. It will never stop growing, so do not wait for FINAL release. This library is used in all of my other projects, as a core for some important features. Please make sure, that you have it installed in order for other libraries to run.

Usage: I do not think that library is so complicated, that it requires documentation, but I have a function list on GitHub.
Features
Image encryption

YOLF library provides great ability to encrypt your images with a given pass code. It supports all usual image formats, but not recommended for *.png files with transparency.

Text encryption

YOLF library has a unique text encryption/decryption algorithms. The greatest thing about it is that you can use your own key phrase for this to work. It is able to work with any files that have raw text (*.txt, *.html, *.css, *.cpp etc.) It has a full support of UTF-8 encoding.

Console costumizations

It never was so easy to costumize the looks of the console window. You now have an ability to resize window, toggle visibility of the scrollbar, and, most importantly, change colors (and even invert them).
